# Pasta de imagens — public/

Adicione aqui as imagens do site. Os nomes abaixo são os que o código já referencia
(troque os arquivos mantendo o mesmo nome, ou ajuste o caminho no componente).

| Arquivo | Onde aparece | Recomendação |
|---|---|---|
| `dra-fernanda-hero.jpg` | Home — topo (hero) | Foto profissional, vertical 4:5, ~1000×1250px. Postura séria e acolhedora, jaleco, fundo limpo azul/branco. |
| `dra-fernanda-clinica.jpg` | Home — bloco "Mais de 20 anos" | Dra. em ambiente clínico, vertical 4:5. |
| `dra-fernanda-retrato.jpg` | Quem Somos — topo | Retrato vertical 4:5. |

## Notas
- Evitar foto "sorriso forçado estilo banco de imagens". Luz natural, fundo neutro.
- Formatos aceitos: `.jpg`, `.png`, `.webp` (webp é o mais leve — recomendado).
- Se trocar a extensão, atualize a referência no arquivo `.tsx` correspondente.
- Coloque também aqui: `favicon.ico`, `og-image.jpg` (1200×630, para compartilhamento).
