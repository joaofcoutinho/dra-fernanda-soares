# dra-fernanda-soares

Website institucional da **Dra. Fernanda Soares** — Dermatologista & Tricologista em Salvador, BA.

Construído com **Next.js 14 (App Router)**, **TypeScript** e **Tailwind CSS**.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse http://localhost:3000

## Páginas

- `/` — Home
- `/quem-somos` — Quem Somos
- `/tratamentos-capilares` — Tratamentos Capilares
- `/tratamentos-dermatologicos` — Dermatologia
- `/localizacao` — Localização
- `/contato` — Contato

## Conteúdo editável

Dados centrais (clínicas, contatos, links) em `lib/site.ts`. Imagens em `public/`.
